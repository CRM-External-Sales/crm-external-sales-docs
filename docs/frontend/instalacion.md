# Instalación (Frontend)

El frontend no se instala por separado: usa el mismo proyecto que la API.

```bash
npm install
npm run dev
```

## Scripts npm

| Script | Acción |
|--------|--------|
| `npm run dev` | Servidor desarrollo (Turbopack) |
| `npm run build` | Build producción |
| `npm run start` | Servidor producción |
| `npm run lint` | ESLint |
